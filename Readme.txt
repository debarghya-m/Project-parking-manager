# Project Parking Manager

Project Parking Manager is a PHP and MySQL-based application designed to manage parking spaces efficiently.

## About

This project is built using PHP for server-side scripting and MySQL for database management. It provides a solution for managing parking spaces, allowing users to efficiently handle parking operations.

## Installation

To set up Project Parking Manager, follow these steps:

### 1. Move Files

Move the `Parking` folder to your XAMPP directory:
xamp->htdocs->paste

### 2. Create Database

Create a MySQL database named `id11637391_parking`:
- Import the provided SQL file into this database.

### 3. Start Servers

Start Apache and MySQL servers in XAMPP application.

### 4. Access Application

Open a web browser and navigate to:
http://localhost/Parking/

## Usage

Once installed and running, you can use Project Parking Manager to manage parking spaces, including adding new vehicles, marking spaces as occupied or vacant, and generating reports on parking usage.
